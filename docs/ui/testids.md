# Haystack UI Test IDs

The EVE-style UI uses stable `data-testid` values for raw Playwright e2e scripts. Dynamic collection rows may repeat a row-level test ID and must expose stable `data-*` attributes for object identity.

| Pattern                                                                                                                                                                                                                          | Applies to                                                                  | Example                     |
| -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | --------------------------------------------------------------------------- | --------------------------- |
| `haystack-app`                                                                                                                                                                                                                   | Root game shell                                                             | `haystack-app`              |
| `world-canvas`                                                                                                                                                                                                                   | React Three Fiber canvas                                                    | `world-canvas`              |
| `boot-screen`                                                                                                                                                                                                                    | Transient boot screen                                                       | `boot-screen`               |
| `neocom`, `neocom-<key>`                                                                                                                                                                                                         | Launcher and six canonical window toggles                                   | `neocom-scanner`            |
| `layout-reset`                                                                                                                                                                                                                   | Per-pilot layout reset                                                      | `layout-reset`              |
| `window-<key>`                                                                                                                                                                                                                   | Window root for `flight`, `scanner`, `cargo`, `comms`, `character`, `bases` | `window-cargo`              |
| `window-<key>-titlebar`, `-icon`, `-label`, `-controls`, `-body`                                                                                                                                                                 | Window chrome parts                                                         | `window-scanner-titlebar`   |
| `window-<key>-minimize`, `window-<key>-close`                                                                                                                                                                                    | Window controls                                                             | `window-cargo-close`        |
| `window-<key>-resize-<dir>`                                                                                                                                                                                                      | Resize handles `n`, `s`, `e`, `w`, `ne`, `nw`, `se`, `sw`                   | `window-scanner-resize-se`  |
| `top-rail`, `rail-metric-<key>`                                                                                                                                                                                                  | Status strip and metrics                                                    | `rail-metric-heat`          |
| `hud-cluster`, `hud-heat-gauge`, `hud-speed`, `hud-velocity`, `hud-position`, `hud-reticle`                                                                                                                                      | HUD telemetry                                                               | `hud-speed`                 |
| `velocity-vector`, `reverse-velocity-vector`, `angular-torque-line`, `angular-torque-pip`, `angular-torque-cap`, `angular-speed-label`, `flight-input-scale-meter`, `flight-input-scale-pip`, `aim-delta-line`, `aim-delta-icon` | Flight vector, angular torque, input scale, and mouse pitch/yaw overlays    | `velocity-vector`           |
| `flight-mode-toggle`, `hud-thrust-<dir>`, `hud-stabilize`, `hud-throttle`, `hud-flight-mode`                                                                                                                                     | Flight controls                                                             | `hud-thrust-fwd`            |
| `scanner-pulse`                                                                                                                                                                                                                  | Scanner pulse control                                                       | `scanner-pulse`             |
| `overview-row`, `overview-col-<field>`, `overview-cell-<field>`                                                                                                                                                                  | Overview table rows, headers, cells                                         | `overview-col-distance`     |
| `overview-filter-<kind>`, `overview-empty`                                                                                                                                                                                       | Overview filters and empty state                                            | `overview-filter-asteroids` |
| `selected-item`, `selected-item-<field>`, `selected-item-stat-<field>`                                                                                                                                                           | Selected Item panel                                                         | `selected-item-name`        |
| `action-<verb>`                                                                                                                                                                                                                  | Selected Item action row                                                    | `action-show-info`          |
| `context-menu`, `context-item-<verb>`                                                                                                                                                                                            | Right-click context menus                                                   | `context-item-show-info`    |
| `scene-bracket`, `waypoint-marker`                                                                                                                                                                                               | DOM scene overlays                                                          | `scene-bracket`             |
| `cargo-capacity-bar`, `cargo-capacity-label`, `cargo-item-<mineral>`, `cargo-empty`, `cargo-sell`                                                                                                                                | Cargo window                                                                | `cargo-sell`                |
| `deposit-list`, `deposit-row-<id>`, `deposit-mine-<id>`                                                                                                                                                                          | Selected asteroid deposits                                                  | `deposit-row-dep-1`         |
| `comms-tab-<channel>`, `comms-log`, `comms-msg`, `comms-input`, `comms-send`, `comms-dm-target`                                                                                                                                  | Comms window                                                                | `comms-tab-global`          |
| `comms-local-list`, `comms-local-<pilotId>`, `comms-local-count`                                                                                                                                                                 | Local pilot list                                                            | `comms-local-count`         |
| `card-row-<pilotId>`, `.pilot-card`, `character-card-org`                                                                                                                                                                        | Character roster                                                            | `card-row-pilot-1`          |
| `showinfo-card`, `show-info-card`, `showinfo-close`                                                                                                                                                                              | Show Info card                                                              | `showinfo-card`             |
| `structure-row-<id>`, `structure-hidden-<id>`, `base-deploy-hab`, `base-field-index`, `field-stats`, `upgrade-<system>`                                                                                                          | Bases and services                                                          | `upgrade-cargo`             |
| `error-pill`                                                                                                                                                                                                                     | Scoped/global error display                                                 | `error-pill`                |

Important raw attributes: `data-open`, `data-focused`, `data-minimized`, `data-z`, `data-min-width`, `data-min-height`, `data-selected`, `data-active`, `data-object-id`, `data-object-kind`, `data-distance-m`, `data-range-m`, `data-busy`, `data-loading`, `data-sort-dir`, `data-speed`, `data-throttle`, `data-flight-input-scale`.
